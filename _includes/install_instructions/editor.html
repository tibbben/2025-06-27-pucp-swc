Editor de texto

Al escribir código, es útil tener un editor de texto optimizado, con funciones como la codificación automática por colores de palabras clave. El editor de texto predeterminado en macOS y Linux suele ser Vim, que no destaca por su intuitividad. Si te quedas atascado, pulsa Esc, seguido de :+Q+! (dos puntos, 'q' minúscula, signo de exclamación) y, a continuación, pulsa Intro para volver a la consola.
